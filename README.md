# Code-and-Data-for-Task-Interleaving-in-Warehouse-Operations
This repository contains the complete replication package—including simulation code, calibration scripts, empirical input data, and output results—for the paper "Task Interleaving in Warehouse Operations: A Discrete-Event Simulation Analysis of Dispatching Rules and Task-Holding Policies".
File Descriptions
sim_interleaving.py

Role: Core Simulation Engine

Description: Contains the primary discrete-event simulation logic and models for warehouse task interleaving (including worker movement, task dispatching rules, and task-holding policy implementations).

experiments_v2.py

Role: Experiment Execution Script

Description: Runs the experimental battery across various parameter combinations, dispatching configurations, and task-holding policies, saving the generated evaluation metrics to exp_results_v2.json.

calibrate_january.py

Role: Model Calibration Script

Description: Processes input data (jan_daily.csv) to calibrate and validate baseline parameters (e.g., travel times, pick rates, order distributions) against real-world daily warehouse operations.

jan_daily.csv

Role: Input Data

Description: Empirical dataset containing daily operational warehouse logs for January used to calibrate baseline model parameters.

exp_results_v2.json

Role: Output Data

Description: Structured JSON file containing raw simulation output metrics and results generated by experiments_v2.py.
